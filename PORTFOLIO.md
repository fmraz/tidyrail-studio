# Product Portfolio

## Selected First Portfolio

### 1. Renewal Desk

- **Value proposition:** Track subscriptions, warranties, renewals, and expiry dates in one private local dashboard.
- **Target audience:** Households, freelancers, office workers, students, and families managing recurring admin.
- **Problem solved:** Important renewals and warranty dates are scattered across inboxes, receipts, notes, and memory.
- **Core features:** Add items, due dates, categories, costs, notes, receipt status, upcoming dashboard, local search, JSON export/import, CSV export.
- **Current free scope:** Local tracker with unlimited practical use, JSON backup/import, CSV export, and no account requirement.
- **Deferred ideas:** Calendar export, saved filters, printable report, and receipt attachment support in a future desktop build.
- **MVP scope:** Static local web app with localStorage, dashboard, item management, search, export/import.
- **v1.0 scope:** Polished UI, full docs, privacy policy, changelog, roadmap, QA, packaged zip.
- **Technical stack:** HTML, CSS, vanilla JavaScript for MVP; optional Tauri/macOS wrapper later.
- **Data storage:** Browser localStorage in MVP; file-backed local data if wrapped as desktop app.
- **Privacy approach:** No accounts, no server, no telemetry, local-only data.
- **Pricing model:** Free-first. No paid downloads, subscriptions, locked Pro features, or artificial limits in the current strategy.
- **Launch channel:** Central Tidyrail Studio website and direct free ZIP download first; Product Hunt or communities only after founder approval.
- **Maintenance complexity:** Low.
- **Risk level:** Low, with clear disclaimers that it is an organizer, not financial advice.

### 2. CleanClip Local

- **Value proposition:** Clean messy copied text into readable notes, emails, and documents without sending it anywhere.
- **Target audience:** Office workers, students, researchers, writers, support teams.
- **Problem solved:** Copied text often includes broken lines, extra spaces, weird bullets, tracking parameters, or inconsistent casing.
- **Core features:** Paste text, clean formatting, remove blank lines, normalize bullets, title/sentence case, URL cleanup, copy result.
- **Current free scope:** Local text cleaner with no upload and no account requirement.
- **Deferred ideas:** Saved cleaning presets, batch mode, keyboard shortcuts, desktop menu bar version.
- **MVP scope:** Browser tool with local-only text processing.
- **v1.0 scope:** Presets, examples, docs, packaged offline app.
- **Technical stack:** Static web app; optional macOS menu bar app later.
- **Data storage:** Optional local presets only.
- **Privacy approach:** Text never leaves device.
- **Pricing model:** Free-first while the product portfolio earns real usage feedback.
- **Launch channel:** GitHub, SEO articles, creator/writer communities.
- **Maintenance complexity:** Low.
- **Risk level:** Low.

- **Value proposition:** Create reusable folder structures for projects, clients, trips, studies, and home admin in seconds.

- **Problem solved:** People repeatedly create the same messy folder trees by hand.
- **Core features:** Template builder, common presets, preview, copy shell commands, downloadable zip structure.
- **Current free scope:** Local template builder with exports and no account requirement.
- **Deferred ideas:** Saved template library, branded client packs, batch generation, desktop file creation.
- **MVP scope:** Browser template builder with export.
- **v1.0 scope:** More templates, polished docs, desktop packaging.
- **Technical stack:** Static web app; optional Tauri wrapper.
- **Data storage:** localStorage template library.
- **Privacy approach:** No uploaded files.
- **Pricing model:** Free-first while the product portfolio earns real usage feedback.
- **Launch channel:** Central Tidyrail Studio website, SEO, and approved community posts.
- **Maintenance complexity:** Low.
- **Risk level:** Low.

### 4. CSV Tidy

- **Value proposition:** Clean small CSV files locally before importing them into spreadsheets, CRMs, or tools.
- **Target audience:** Operators, freelancers, small businesses, assistants, researchers.
- **Problem solved:** CSV files often have broken headers, whitespace, duplicate rows, inconsistent delimiters, and encoding issues.
- **Core features:** Load local CSV, preview, trim whitespace, normalize headers, remove duplicates, export cleaned file.
- **Current free scope:** Local CSV cleaner for small files, with clear safety warnings and no upload.
- **Deferred ideas:** Batch cleaning, saved recipes, column transforms.
- **MVP scope:** Local browser CSV cleaner for non-sensitive small files.
- **v1.0 scope:** Strong warnings, no upload, QA with sample files.
- **Technical stack:** Static web app with built-in parser or lightweight dependency only if needed.
- **Data storage:** In-memory only unless recipes are saved.
- **Privacy approach:** File processed locally in browser.
- **Pricing model:** Free-first while the product portfolio earns real usage feedback.
- **Launch channel:** SEO, small business communities, GitHub free version.
- **Maintenance complexity:** Medium.
- **Risk level:** Medium due to CSV edge cases.

### 5. Packwise Lists

- **Value proposition:** Generate and reuse practical packing checklists for trips, family visits, moves, and events.
- **Target audience:** Families, students, frequent travelers, event organizers.
- **Problem solved:** Packing and preparation lists are rebuilt from memory and easily miss small items.
- **Core features:** Trip profile, list templates, custom items, reusable lists, print/export.
- **Current free scope:** Local checklist generator with reusable lists and export/print support where practical.
- **Deferred ideas:** Saved template packs, household profiles, printable PDFs.
- **MVP scope:** Local checklist generator.
- **v1.0 scope:** Polished templates, print views, docs.
- **Technical stack:** Static web app.
- **Data storage:** localStorage.
- **Privacy approach:** No account, no travel data upload.
- **Pricing model:** Free-first while the product portfolio earns real usage feedback.
- **Launch channel:** SEO and family/travel communities.
- **Maintenance complexity:** Low.
- **Risk level:** Low.
