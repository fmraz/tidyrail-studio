# Launch Checklist

Use this checklist before any public launch. Public release requires explicit founder approval for the concrete destination and action.

## Product Readiness

- Working MVP.
- Tested core flow.
- Clear UI and empty states.
- README with install/run/build instructions.
- Privacy policy draft.
- Product description.
- Pricing recommendation.
- Landing page copy.
- Launch copy.
- FAQ.
- Changelog.
- Roadmap.
- Support template.
- Known limitations.
- Release checklist completed.

## External Action Approval

Require explicit approval for:

- Buying domains, hosting, ads, templates, fonts, licenses, APIs, or marketplace listings.
- Registering accounts.
- Publishing publicly.
- Posting on social media.
- Sending email.
- Submitting to marketplaces or directories.
- Entering payment details.
- Confirming paid purchases.
