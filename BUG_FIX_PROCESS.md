# Bug Fix Process

## Intake

Collect:

- product
- version
- platform
- browser or OS
- steps to reproduce
- expected behavior
- actual behavior
- screenshots or console errors if available

## Triage

Rank by:

- data loss risk
- security/privacy impact
- affected users
- workaround availability
- release blocker status

## Fix

- Write or update a focused test/check where practical.
- Keep patch scope small.
- Update known limitations if the issue is not fixed immediately.
