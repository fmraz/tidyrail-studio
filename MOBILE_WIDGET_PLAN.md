# Mobile Widget Plan

Status: planned after mobile apps.

## Renewal Desk Widget Value

Useful widget ideas:

- Next renewal due
- Renewal count due in the next 30 days
- Quick add reminder action
- Receipt missing count

## iOS

Use WidgetKit after the iOS app exists. Requires Apple Developer membership before device distribution and App Store submission.

## Android

Use Android App Widgets after the Android app exists. Requires Google Play setup for store distribution.

## Rule

Do not build decorative widgets. Only ship a widget if it provides a quick status or action that users would reasonably check from the home screen.
