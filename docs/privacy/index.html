Privacy policy.

Tidyrail Studio is designed around minimal data collection, local-first products, and user-controlled exports.

Last updated: June 28, 2026

Current products

Renewal Desk stores records locally in the browser in the current free version. Tidyrail Studio does not receive those local records unless a future cloud sync feature is explicitly enabled by the user.

Future accounts

Cloud sync is not active in the current public release. If accounts are enabled later, this policy will be updated before launch and the account system should store only the information required for authentication, sync, export, deletion, and privacy settings.

Analytics

The current public website does not include advertising pixels or invasive analytics scripts. If privacy-friendly analytics are added later, this policy will be updated before use.

Exports and deletion

Users should be able to export their product data and delete local or cloud data when the relevant storage mode is active.

Security

Security posture, vulnerability reporting, and current HTTPS status are published on the security page.
