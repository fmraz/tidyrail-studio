# Risk Register

| Risk | Severity | Mitigation |
|---|---|---|
| Company name later conflicts with trademark or domain availability | Medium | Run formal checks before public launch or domain purchase |
| LocalStorage users clear browser data and lose app data | Medium | Prominent export/backup copy and JSON export/import |
| Product scope expands before first release | High | Keep one primary focus per 3-hour cycle and finish Renewal Desk first |
| Marketing copy overpromises privacy or outcomes | Medium | Keep copy factual and specific |
| CSV/PDF/file tools create edge-case support burden | Medium | Start with simpler products and document limitations |
| Paid packaging requires marketplace, licensing, or payment setup | Medium | Prepare assets only; ask approval before external steps |
| User data sensitivity | Medium | Prefer local-first and no telemetry; avoid uploads |
