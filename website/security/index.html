Security posture.

Tidyrail Studio keeps the current release intentionally small: no ad tracking, no unnecessary accounts, and clear controls for user data.

Current website baseline

The public site is designed to keep the attack surface small. HTTPS is being finalized for the custom domain before it is enforced.

Security controls

  • Limited page accessPages are restricted to the resources they need.
  • User-controlled recordsRenewal Desk keeps records on this device and lets users download a copy.
  • No hidden account systemAccount sync will not launch until privacy and deletion controls are ready.
  • No ad trackingThe public release does not include advertising networks or invasive analytics.

Report a vulnerability

Please avoid posting sensitive exploit details publicly. For non-sensitive product issues, use the public repository issue tracker. A dedicated security email will be published only after mail handling for the domain is ready.
