Privacy

Privacy policy.

Tidyrail Studio avoids unnecessary tracking, accounts, and data collection.

Current products

Renewal Desk stores records locally in the browser in the current free version. Tidyrail Studio does not receive those records.

Analytics

No analytics scripts are included in the current static website draft or Renewal Desk release candidate.

Exports

Exported JSON and CSV files are controlled by the user and may contain information entered into the app.

Changes

This policy will be updated if a future product adds optional sync, accounts, analytics, or other network features.
