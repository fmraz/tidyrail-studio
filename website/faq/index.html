FAQ

Questions and clear answers.

Are the products free?

Yes. The current strategy is free products first. No paid downloads, subscriptions, or locked Pro features are implemented now.

Does Renewal Desk require an account?

No. No. You can use the current version in your browser without signing in.

Where is Renewal Desk data stored?

Your list stays on the device and browser where you use the app. You can download a backup whenever you want.

Does Tidyrail Studio track users?

No advertising tracking or invasive analytics are included in the current public site or Renewal Desk release.

How do I report a security issue?

Use the security page for reporting guidance. Do not post sensitive exploit details in a public issue.

Will there be desktop and mobile apps?

Yes. Desktop and mobile versions are planned after the web app is stable and useful for real users.
