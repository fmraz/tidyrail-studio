# Renewal Desk 0.1.0 MVP

Renewal Desk is a free, local-first tracker for subscriptions, warranties, service dates, and document expiry reminders.

## Included

- Add, edit, search, filter, renew, and delete tracked items.
- Dashboard summaries, upcoming dates, receipt status, and a calendar view.
- Monthly, quarterly, and yearly renewal advancement with Undo.
- Versioned JSON backup and restore with validation and recovery.
- Spreadsheet and calendar exports.
- Responsive desktop and mobile browser layouts.
- Installable PWA shell with offline reload after the first successful visit.

## Data and Privacy

No account is required. Records stay in browser storage on the device where they were created unless the user exports a backup. Renewal Desk does not include advertising pixels or invasive analytics.

## Known Limitations

- Records do not sync between devices yet.
- Browser storage can be cleared by browser or device settings; regular backups are recommended.
- Currency formatting currently uses USD.
- One-time and custom schedules require manual next-date updates.
- Native desktop and mobile installers are not included in this release.

## Installation

Use the web app from the Tidyrail Studio website, or download the static ZIP and open `index.html` in a modern browser.
