# Privacy Policy Draft

Status: draft, not legal advice.

Effective date: draft.

Tidyrail Studio builds free productivity tools with a privacy-first approach.

## Current Public Website

The current website is static and does not intentionally use advertising trackers or invasive analytics.

## Renewal Desk Current Release

The current Renewal Desk MVP stores data locally in the user's browser. Data can be exported as JSON or CSV from the app.

## Future Accounts

When accounts are enabled, Tidyrail Studio plans to store:

- email address for authentication,
- optional display name,
- product data explicitly saved by the user,
- privacy and locale preferences.

## Data Rights

Users should be able to:

- export their data,
- delete local data,
- request cloud account deletion,
- delete product data,
- manage privacy preferences.

## No Sale of Data

Tidyrail Studio will not sell user data.

## Review Needed

This document must be reviewed before public account launch.
