# Renewal Desk App Icon Spec

## Concept

A translucent renewal loop wraps a small document tile with a subtle amber reminder point.

Meaning:

- Renewal loop: recurring services and reminders.
- Document tile: warranties, subscriptions, service records, and expiry dates.
- Amber point: upcoming deadline.

## Constraints

- No text in final app icon.
- No platform logos.
- No calendar design copied from any OS.
- Must remain legible at small sizes.
- Should work on light and dark backgrounds.

## Production Tasks

1. Refine vector mark.
2. Export full icon set.
3. Test on macOS dock, Windows taskbar, Linux launcher, iOS home screen, Android launcher, and browser tabs.
4. Prepare store icon PNG.
