# Icon System

Status: concept created; production exports still needed.

## Style

The Tidyrail Studio app icon family should use:

- Rounded-square base.
- Liquid-glass surface treatment.
- Simple central symbol.
- Teal, graphite, icy blue, and restrained amber accent.
- Clear silhouette at 16px.
- No platform logos or copyrighted icon metaphors.

## Company Mark

Current mark: simple `T` block for early website launch.

Future direction:

- Create a vector monogram based on a rail/path metaphor.
- Export SVG, favicon, and social preview variants.
